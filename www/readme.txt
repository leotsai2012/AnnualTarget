To switch between mock and real cordova.js:
1. open index.html;
2. add/remove ".mock" to cordova.js script reference;
3. refresh your browser.

What is cordova.mock.js? Please refer to my blog post:
http://apachecordova.blogspot.com/2013/08/phonegap-development-best-practice1.html


Steps to run this demo:
1. deploy this website to IIS(web server);
2. open google chrome web browser (SQLite will not work on firefox or IE);
3. enter url something like: http://localhost:8090
4. done. You can zoom the browser to a size of phone, refresh.

If you want to build it to android and iOS, you need to follow the guide on http://docs.phonegap.com.
